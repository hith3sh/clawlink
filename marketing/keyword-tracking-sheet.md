# ClawLink Keyword Tracking Sheet

Purpose: track the small set of queries that actually matter for ClawLink/OpenClaw discovery, content planning, and internal linking.

Use this as a working SEO sheet, not a fake-enterprise dashboard.

## How to use this

Update these fields when possible:

- **Target URL** — the page that should rank for the query
- **Stage** — `live`, `drafting`, `planned`, or `needs-work`
- **Intent** — `troubleshooting`, `integration`, `workflow`, `comparison`, or `brand`
- **Priority** — `P1`, `P2`, or `P3`
- **Index status** — `yes`, `no`, or `check`
- **Notes** — use for internal link opportunities, title tweaks, or missing page ideas

Suggested review cadence:

- quick pass once per week
- deeper cleanup once per month

## Internal linking ideas

Use this list when updating existing pages:

- From every integration guide, link to:
  - `/openclaw`
  - `/troubleshooting`
  - at least one workflow/use-case page
- From troubleshooting pages, link sideways to:
  - related ACP/ACPX pages
  - `/openclaw`
  - relevant integration pages when the failure blocks setup
- From workflow pages, link down to:
  - exact integration setup guides
  - billing/security if trust objections matter
  - troubleshooting pages when common setup failure exists
